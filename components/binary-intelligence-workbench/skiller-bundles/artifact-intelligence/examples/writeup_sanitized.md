# Sanitized Example: Professional Writeup Generation

A finished case should produce a report that separates observed evidence, technical analysis, verification, and remediation or lessons learned.

Recommended structure:
- Executive summary
- Artifact inventory and hashes
- Tooling/environment
- Investigation timeline
- Technical deep dive
- Failed paths and pivots
- Verification
- IOCs/remediation when applicable
- Redaction notes

Standalone dark-mode HTML is useful for local viewing and sharing in a controlled environment.
